  • INTRO WEB


    • Definisi Internet, merupakan jaringan komputer yang sangat besar. Internet menggunakan jaringan komunikasi di seluruh dunia. Internet juga menghubungkan jaringan besar ke jaringan kecil. Protokol TCP/IP.

    Fasilitas Sebuah Internet, WEB, Mail, Chat, dan File Transfer.

    Ciri Aplikasi Berbasis Internet :
    1. Menggunakan protokol aplikasi yang sudah ada atau mendefinisikan protokol sendiri.
    2. Aplikasi di sisi server berkomunikasi langsung dengan client.
    3. Aplikasi di sisi client dapat berupa aplikasi yang berdiri sendiri atau temple dalam aplikasi lain.

    Ciri Aplikasi Berbasis Web :
    1. Menggunakan protocol HTTP.
    2. Aplikasi di sisi server berkomunikasi dengan client melalui web server.
    3. Aplikasi di sisi client umumnya berupa web browser.

  • CLIENT SIDE PROGRAMMING


    • Client side pemrogramming biasanya lebih dikerjakan oleh fornt-end developer. Komponen-komponen client side programming adalah HTML, CSS, dan Javascript.

    HTML (Hyper Text Markup Language) adalah standard bahasa yang digunakan untuk menampilkan dokumen web.
  • Fungsi :
    • 1. Mengontrol tampilan konten.
    2. Memuat form untuk transaksi data.
    3. Memuat obyek lain seperti image, audio, video, dll.

    HTML TOOLS :
  • Browser
    • 1. Software di client untuk menerjemahkan tag HTML (Halaman Web)
    2. IE, Chrome, Firefox, Opera, dll.
  • Editor
    • 1. Program untuk menulis tag HTML.
    2. Notepad, Notepad ++, Sublime, Dreamweaver, dll.

    TAG-TAG HTML :
    Di bahasa pemrograman pada umumnya biasanya kita mengenal code, pada HTML kita mengenal tag.
  • Contoh Tag
    • < body>, < head>, < div>, < img>, dll.
  • Tag tidak case sensitive. Artinya < body> == < BODY>.
